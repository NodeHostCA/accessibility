Adding accessibility to any website

This is a easy to embed script that helps make websites more accessible and easer to use with things like font size changing, contrast settings, easy to access menu, and more. This will NOT replace standard site changes that everyone should make, but helps with making designs easer based on user needs. For example the ability to make text high contrast by turning the page into almost a reader mode. We also have options like making forms easer to see and modify inputs by highlighting content.

You can test by tapping the tilde key or escape on your keyboard.
